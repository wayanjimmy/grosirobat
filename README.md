## Panduan Pengembangan

1. Install [Docker](https://docs.docker.com/install/#supported-platforms) sesuai dengan OS.
2. `git clone https://gitlab.com/wayanjimmy/grosirobat.git`
3. `cd grosirobat && docker-compose up`
